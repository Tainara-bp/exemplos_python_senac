# 📚 Verificador de Aula

Este é um programa simples em Python que verifica se haverá aula dependendo do período informado pelo usuário.

## 🚀 Funcionalidade

O programa funciona da seguinte forma:

- Se o usuário digitar **manhã**, o sistema informa que **tem aula**.
- Se o usuário digitar **tarde**, o sistema informa que **não tem aula**.
